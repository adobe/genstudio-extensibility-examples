# Components Folder

This folder contains the custom UI components for the External DAM App.

## Files

- **AssetViewer.tsx** - Main panel component for browsing and selecting assets from external DAM
- **AssetCard.tsx** - Individual asset card component displaying asset information

## Customization

Feel free to modify these components to match your design requirements and business logic. These are the primary files you should edit to customize the user interface and functionality of the asset selection experience.
