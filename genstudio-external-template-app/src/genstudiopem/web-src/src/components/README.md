# Components Folder

This folder contains the custom UI components for the External Template App.

## Files

- **TemplateViewer.tsx** - Main panel component for browsing and selecting templates
- **TemplateCard.tsx** - Individual template card component displaying template information

## Customization

Feel free to modify these components to match your design requirements and business logic. These are the primary files you should edit to customize the user interface and functionality of the template selection experience.
